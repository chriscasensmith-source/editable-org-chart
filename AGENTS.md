# Project instructions

Goal:
Build an editable org chart tool from a single source data table.

Project rules:
Use the existing org charts as reference files only.
Create one editable source table for the real data.
Support two scenarios: current and future.
Show reporting lines clearly.
Make vacant roles easy to spot.
Keep the interface simple and business friendly.
Make it easy to print or export.

Preferred structure:
Use a single data table with fields like:
role_id
scenario
employee_name
title
manager_role_id
department
state
role_status
notes

Definition of done:
A user can update the data without redrawing the org chart.
The page can switch between Current State and Future State.
Vacant roles stand out clearly.
The result is easy to print and present.
Add a short explanation of how to update the data later.
